AuthentiCMD
===========

Command line tool to read the public key of an Authenticode signed filed

Download
--------

[AuthentiCMD.zip](bin/AuthentiCMD.zip?raw=true)

Example
-------

Input:
```shell
AuthentiCMD.exe c:\windows\explorer.exe
```
